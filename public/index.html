Jenkins Sucks

Change my mind.

This simple API can be used to send Rate-Limited requests to a Jenkins (?) of your choosing.

How To Use Me

Authentication

To make successful requests to your Jenkins endpoints, you must provide a valid Jenkins API token.

The Header

Send the token in a x-jenkins-token header. If this weirds you out, take my word for it, we're not storing these. Or don't--honestly I don't care.

Make the Request

Simply make a GET or POST request that contains the Jenkins Base URL (and port, if necessary) and Job Name to the /generic-job endpoint.

Example

POST https://jenkins-sucks.heroku.com/my.base.url:1337/MyJobName
